## Overview

Turn your scattered work into organized weekly reports. Perfect for engineers, managers, and anyone who needs to answer "what did I do this week?" with actual data.

### Key Features

- **Beautiful CLI** with Rich formatting and progress bars
- **GitHub Integration** for tracking commits, PRs, and reviews
- **Calendar Integration** with Google Workspace
- **Multiple Output Formats** (Markdown, HTML, JSON)
- **Render Engine** with format conversion and caching
- **Display Mode** for viewing reports in terminal
- **AI Activity Tracking** for Claude/AI-assisted development

## Quick Start

```bash
# Install with uv (recommended)
uv pip install what-did-i-get-done-this-week

# Setup
receipts setup

# Generate report
receipts
```
